Welcome to stalker’s documentation!¶

Stalker is an OpenSource Production Digital Asset Manager (ProdAM) licensed under GPL v3 and designed specially for Animation and VFX Studios.

Stalker consists of several parts:
  • Database
  • Extensions
  • Models
  • UI

The design of the Stalker is tried to be kept as flexible as possible.

Stalker is build over these other OpenSource projects:
  • Python
  • SQLAlchemy
  • PyQt
  • JQuery
  • Jinja
